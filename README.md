# Gas Stock Management

Next.js 14 + MongoDB dashboard for LPG cylinder distribution businesses.

## Features
- Single-user JWT auth with httpOnly cookies
- Customer, Inventory, Purchase, Delivery CRUD
- Automatic stock increase/decrease on purchase/delivery
- Dashboard KPIs and charts
- Profit & loss summary
- CSV report export
- Mobile-friendly dashboard layout

## Setup
```bash
npm install
cp .env.example .env.local
npm run dev
```

## Environment Variables
See `.env.example`.
